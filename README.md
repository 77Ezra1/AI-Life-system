# AI-Life-system
个人AI生活系统

## 项目简介

这是一个个人AI生活系统项目，旨在构建一套完整的AI辅助工具集。

## 子项目

### ai-chat-app
基于Apple设计理念的AI对话应用前端界面

**特性：**
- 🎨 遵循Apple Human Interface Guidelines设计规范
- 💬 支持多AI模型配置（OpenAI、Anthropic、Google）
- 🌓 深色/浅色主题切换
- ✨ 毛玻璃效果（Liquid Glass）
- 📱 响应式设计
- ⚡ 流畅的动画和交互

**技术栈：** React 18 + Vite + Tailwind CSS + shadcn/ui

[查看详细文档](./ai-chat-app/README.md)
